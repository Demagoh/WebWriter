# WebWriter
**WebWriter** is a small-ish web-based writing application, intended to work as a simple text editor.<br />
It is currently still in development.

## Functionality
I currently plan to add the following functionalities:
- simple text editing
    - I currently do not plan on adding any keyboard shortcuts
- text styling
    - most probably using Markdown

<br />

I would also like to add the following functionalities at some point (if I can be bothered to actually figure out how to make them work):
- collaborative editing
    - think Google Docs but worse
- version control (using Git)
    - I'm not too sure about this one, but it would be a cool addition
- media support
    - I imagine that handling adding images, videos and audio clips will be quite hard to do, with my first web-based application (Gyped) I resorted to never actually displaying them to the user directly due to CSS styling issues

## Contributing
I'm making this app for myself because I want a text editor I can access and have work *almost* the same on all of my devices. It is because of this that I don't intend on letting other people contribute their own code to this project.<br />
<br />
Instead, I would like to encourage anybody who would like to add something to this application to create a <a href="https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo" target="_blank">fork</a> of this repository and then work on the fork.<br />

## Usage
If you'd like to try out this app you can do so by setting up your own instance of the app on your own web server.<br />
I plan to create a guide on how to set it up once the first full version of the app is finished.<br />
Before then you can check out <a href="https://demagoh.com/about/thehow" target="_blank">this "guide"</a> on setting up a web server I created some time ago.
