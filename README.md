# WebWriter
**WebWriter** is a small-ish web-based writing application, intended to work as a simple **text** editor.<br />
It is currently still in <a href="https://github.com/Demagoh/WebWriter/tree/development">development</a>.

## Features
I currently plan to implement the following features:
- simple text editing
    - keyboard shortcuts might be added later
- text styling
    - either Markdown or something custom
- configuring the application
    - if and how people can create accounts to access the application
    - (when implemented) if and how people can collaborate
    - (when implemented) if and what sizes/typed of images people can upload

<br />

I would also like to implement the features listed bellow at some point (if I can be bothered to actually figure out the implementations):
- collaborative editing
    - like Google Docs but slightly worse
- media support
    - while this application is only really meant to be used as a **text** editor, I do think it'd be nice to be able to add images to the text, *if you have the storage for them*
- ~~version control (using Git)~~ (dismissed)
    - decided against this because I can tell there'll be complications with integrating version control with a database

## How to use the application
If you'd like to try out this application you can do so by setting up your own instance of it on your own web server.<br />
I plan to create a guide on how to set it up once the first full version is finished.<br />
While you wait for the first version to release you can check out <a href="https://demagoh.com/about/thehow" target="_blank">this "guide" on setting up a web server</a> I wrote (and now maintain).
